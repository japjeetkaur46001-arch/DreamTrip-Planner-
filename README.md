# Dream Trip Planner

## Project Description
Dream Trip Planner is a Flask and PostgreSQL based web application that helps users manage their dream travel destinations.

## Features
- Connects with PostgreSQL Database
- Stores destination details
- Displays all destinations
- Easy to expand with Add, Edit and Delete features

## Technologies Used
- Python
- Flask
- PostgreSQL
- psycopg2
- HTML
- CSS

## How to Run

1. Install Python
2. Install PostgreSQL
3. Install required packages

```bash
pip install -r requirements.txt
```

Run the project

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
